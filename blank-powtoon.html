28. Answer this for 100 Pts. List down the different Toolbar tools in Powtoon and give the uses/description of each.


Naming Your Powtoon

By default, your Powtoon will be named Untitled-x, where x is the running number of the Powtoon that you have created.

You can rename the Powtoon in the following locations:

In the Powtoon Studio

Select the text in the Edit Powtoon title field and then type the new name.

On the 'My Powtoons' page

a. Locate the Powtoon that you want to rename.

b. Select the current name and then type the new name.

c. Click the checkmark.

Saving Your Powtoon

Saving as you work

To save your Powtoon as you work, click SAVE at the top of the screen.

Note:

Your Powtoon is also saved automatically every 2 minutes.

Saving a copy of your Powtoon

To create a copy of the Powtoon that you are working on, use the Save a Copy functionality.

Select the current Powtoon title.

Type a name for the new Powtoon.

Click Save a Copy on the right side of the screen.

Save a Copy will create a new copy of the Powtoon and display it on the stage.

The original Powtoon (that was copied) will be saved in 'My Powtoons'.
