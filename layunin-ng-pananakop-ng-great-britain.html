3. dahilan ng pananakop ng great britain​


Paano nasakop ng Britain ang India?

Nagawa ng Britain na masakop ang malaking bahagi ng India pagkaraan ng Labanan saPlassey noong 1757. Ang pangkat- etnikong Maratha ang pangunahing tutol sapagpasok ng mga British, subalit nagapi ito ng mga Briitsh noong 1803. Tuluyan nangnasakop ng Britain ang India noong ika-19 na siglo.

Ang mga British ay nakasakop sa malaking bahagi ng India sa pamamgitan ng "Divide and Rule Policy"na kung saan pag-aaway-awayin nila ang mga magkakalabangkaharian sa India. Sa pamamagitan ng pagkontrol ng mga British sa kalakalan ng Indiaat paglakas ng kapangyarihan nito, pinairal nila ang patakarang pampolitika, pangekonomiya at panlipunan.

Mga Karanasan ng India sa Panahon ng Pananakop ng Britain:

Ang kanilang pananakop ay nagresulta sa pagkatatag nila ng British East IndiaCompany. Ito ay naghanap ng mga produkto tulad ng pampalasa tulad ng paminta,nutmeg at cloves. Ang spices ay itinuturi ng Europeo bilang kayamanan. Tinawag nilang “Spice Island” ang India, kasama ng Moluccas at Sri Lanka. Maliban sa mga pampalasa, higit na pinagkakitaan ng kompanya ang mga tela, lalo na ang cotton.

Nagkaroon rin ng konstruksiyon ng mga riles ng tren upang mapabilis ang daloy ngtransportasyon at kalakalan. Naisakatuparan naman ito sa pamamagitan ngpagtratrabaho ng mga katutubo. Ipinagbawal ang ilang tradisyong Hindu gaya ngSuttee. Dahil sa makabagong paraan ng transportasyon at teknolohiya, napag- isa angmga Indian. Ipinakilala ng mga dayuhang mananakop ang maayos na sistema ngpamamahala ng bansa. Pinasimulan rin ang sistema ng pagbabangko at nagtayo din ngmga gusali upang mapakinabangan ang mga likas na yaman ng bansa.

23. epekto ng pananakop great britain?​


Masamang Epekto

Maraming masamang epekto ang pananakop ng mga Briton (Gran Britanya) sa bansang India. Ilan sa mga naging epekto nito ay ang malawakang di-pagkakapantay-pantay, kawalan ng pagpaparaya ng mga tao sa iba’t ibang paniniwala at relihiyon, ang pagtingkad ng caste system, at ang pagkakawatak-watak ng India na naging sanhi ng pagsibol ng mga bansang Pakistan at Bangladesh.
