10. Gilbert has 95 rose bushes to plant in rows. He wants to put the same number of bushes in each row with no bushes left over. How many rows of bushes could Gilbert make?


Answer:

To find the number of rows of bushes that Gilbert could make with 95 rose bushes, you can divide the number of bushes by the number of bushes per row to find the number of rows.

Since Gilbert wants to put the same number of bushes in each row with no bushes left over, he will need to find the greatest whole number that is less than or equal to 95 and divides evenly into 95.

One way to do this is to start with the highest possible value and work your way down, checking to see if it divides evenly into 95. For example, if Gilbert starts with 95 bushes per row, he would have 1 row of 95 bushes. If he tries 94 bushes per row, he would have 95 / 94 = 1.010638298, which is not a whole number, so 94 bushes per row would not work. If he tries 93 bushes per row, he would have 95 / 93 = 1.034482759, which is also not a whole number. However, when he tries 92 bushes per row, he has 95 / 92 = 1.038043478, which is a whole number. Therefore, Gilbert could make 1.038043478 rows of 92 bushes each.

Therefore, Gilbert could make 1 row of 92 bushes.
