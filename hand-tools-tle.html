3. How are hand tools properly stored? TLE 8


Answer:

Store short-handled tools in a plastic bin or box. Tie together long-handled tools in a garbage can while in storage. Otherwise hang these tools on a wall. If you are storing tools in self storage, place small tools in plastic bins.

The following is a statement on how to take care of measuring tools:

1. Check the functionality of the measuring tool regularly ✔

This will be very useful for maintaining measuring tools so that they are durable and can be used for a long time.

2. Lubricate movable parts of measuring tools ✔

This will prevent the movable parts of the measuring tools from stuck so that they cannot move.

3. Wipe off measuring tools that are made out of thin steel sheets with bare hands ✘

Measuring tools made out of thin steel are usually sharp, so wiping them with your bare hands can injure your hand.

4. Exposing measuring tool to rough particles is fine ✘

In some types of measuring tools, this can cause damage to the measuring tools and reduce the accuracy of measurement results.

5. Regularly clean measuring tools and their storage ✔

This will be very useful for maintaining measuring tools so that they are durable and can be used for a long time.

19. Directions: Read and understand the statement. You will be graded through thefollowing rubrics. Write your answer in your TLE CSS Activitynotebook1. What is the importance of knowing the appropriate hand toolsfor Computer System Servicing?​


It is very important to know how to handle different tools when you are studying and teaching computer system servicing. Like any other course, tools are helpful in performing your job and course effectively. Tools like crimping tool, a known networking tool that could trim, cut and crimpped cables to Rj45's must be handled with care. The crimping tool can attached Rj45's to UTP cables. If it's not handled properly, it can damaged the cables or the computer, worse, it can cause injuries to the user. Crimping tools has small sharp blades that could cut through wires or wire covers. Other tools like PPE's are also necessary to know how to handle and use them.
