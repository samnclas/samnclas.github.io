1. erase is one of the ms paint icons which is used to


Answer:

Eraser Tool

Eraser Tool

This tool is used to remove parts of the active layer or selection. It can be useful when working with multi-layered images as portions of an upper layer erased will allow lower layers to show through.

The Eraser works by marking areas of the layer as being transparent (i.e. having an Alpha value of zero) or partially transparent (Alpha < 255). Areas completely erased will have the original RGB color information replaced with black (hexadecimal code #000000). Areas undergoing partial erasure will retain the original color information and have a lower opacity value applied.

The Eraser inherits the Brush Width and Hardness settings from the Tool Bar. The Brush Width determines how wide the circular tip of the Eraser will be. Note that the tip of the Eraser will always be circular - it is not possible to alter this.Hardness indicates how hard or soft the erased edges are.

A low setting gives a softer or more diffuse edge, while a high setting gives a harder or more well defined edge.The Eraser inherits the Alpha value of the Primary color if the Left Mouse button is used. It inherits the Alpha value of the Secondary color if the Right Mouse button is used. This makes it possible to make the 'soft' erasings.Eraser Tool showing three Alpha settings.On the left, the Alpha Setting is set to 60. The opacity of the erased area will be 255-60=195 and the original color information will be retained.In the middle the Alpha Setting is set to 120. The opacity of the erased area will be 255-120=135. Again, the original color information will be retained.On the right, the Alpha Setting is set to 255 (maximum). The opacity of the erased area will be 255-255=0. The original color information will be replaced with black (#000000)
