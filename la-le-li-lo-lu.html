Using the Pythagorean theorem and basic geometry principles, we can find the missing values:

Right angle O: Since angle L is a right angle, we know that angle ULO is also a right angle. Therefore, angle OLU is also a right angle, and angle O is a right angle as it is complementary to angle OLU.

EX: Using the Pythagorean theorem, we can find the length of EX:

EL^2 + LX^2 = EX^2

Substituting the given values, we have:

38^2 + 18^2 = EX^2

1444 + 324 = EX^2

1768 = EX^2

EX = sqrt(1768) ≈ 42.05

LE: We can find the length of LE using the Pythagorean theorem:

EL^2 + LX^2 = EX^2

EL = sqrt(82.60) ≈ 9.09

19. Question given: Quadrilateral LIKE is a kite with line LI≈LK and line LE ≈KE. A. LE is twice LI. If it's perimeter is 21 cm, how long is line LE?


Let LI be x and LE is 2x.
Perimeter = 21 cm

Perimeter of kite LIKE = 2 (LI) + 2 (LE)
     21 = 2 (x) + 2 (2x)
     21 = 2x + 4x
     21 = 6x
    
     6x/6 =21/6
     x = 7/2

Plug-in 7/2 to x:
    LI = x          ⇒   LI = 7/2 cm
    LE=2x        ⇒    LE = 2(7/2) = 7 cm

ANSWER: LE is 7 cm.

Check: 
     7/2 + 7/2 + 7 + 7 = 21
     14/2 + 14 = 21
      7 + 14 = 21
      21 = 21   (True)

I have encountered this problem before. This was my answer.

Given:
LI is congruent to IK ⇒ LI = IK
LE is congruent to KE ⇒ LE = KE
LE is twice LI ⇒ LE = 2LI

Perimeter = LI + IK + LE + KE
21 cm = 2LI + 2(2LI)
21 cm = 6LI
21 cm/6 = LI
3.5 cm = LI

LI = IK : LI = 3.5 cm ; IK = 3.5 cm
LE = KE: LE = 2LI ; LE = 2(3.5 cm) ; LE = 7 cm; KE = 7cm

Perimeter = LI + IK + LE + KE
21 cm = 3.5 cm + 3.5 cm + 7 cm + 7 cm
21 cm = 21 cm
