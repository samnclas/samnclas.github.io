30. This HTML formatting tag is used to display an important text. (x^2+2x − 63)÷(x+9)


Answer:

x-7

Step-by-step explanation:

The simplest way to divide these polynomials is to format them as [tex]\frac{x^{2} +2x-63}{x+9}[/tex].

Then factor the numerator: x²+2x-63=(x+9)(x-7), so [tex]\frac{x^{2} +2x-63}{x+9}=\frac{(x+9)(x-7)}{x+9}[/tex].

Dividing the (x+9) in the numerator by x+9 in the denominator leaves you with x-7.
