21. 4+3=16 first degree equation​


Answer:

3x = 16 is x = 4.

Step-by-step explanation:

4 + 3x = 16

To solve for x, we need to isolate it on one side of the equation. We can do this by subtracting 4 from both sides of the equation:

4 + 3x - 4 = 16 - 4

Simplifying:

3x = 12

Finally, we can solve for x by dividing both sides of the equation by 3:

3x / 3 = 12 / 3

x = 4

Therefore, the solution to the equation 4 + 3x = 16 is x = 4.
