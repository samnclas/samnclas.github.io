8. mga salitang magkasalungat sa pangungusap​


Answer:

Ang magkasalungat o kasalungat ay mga salitang baliktad ang kahulugan sa isa't isa.

Halimbawa

Madali-Mahirap

Pangungusap: Akala ko ay madali lamang ang pagsusulit, mahirap pala.

Mainit-Malamig

Pangungusap: Nagtimpla ako ng mainit na kape dahil malamig ang panahon.

Mabagal-Mabilis

Pangungusap: Mabagal kumilos ang orasan kung iyong babantayan ngunit ito'y mabilis kung hindi mo mamalayan.

Mataas-Mababa

Pangungusap: Mataas ang nakuhang marka ni Sisa sa Filipino ngunit mababa naman pagdating sa Matematika.

Maingay-Tahimik

Pangungusap: Maingay ang klase kapag umalis ang guro, tahimik naman kapag nandiyan.

MGA SALITANG MAG KASALUNGAT

MALI :: TAMA

Si mark ay nag tatapon ng basura sa kalsada, nakita ito ng kanyang mga magulang at sinabing mali ang kanyang ginagawa kayaʼt tinuruan siya kung paano ang tama na paraan ng pag tapon ng basura

UMAGA :: GABI

Sa korea ay umaga na at dito naman sa pilipinas ay gabi pa.

MAPUTI :: MAITIM

Ang balat ng kanyang ina ay maputi, samantala ang balat ng kanyang anak ay maitim.

MARUMI :: MALINIS

Marumi ang kanyang pananamit ngunit ang kanyang matalik na kaibigan ay malinis manamit.

MALUNGKOT :: MASAYA

Malungkot si eisha dahil hindi niya nagustuhan ang kanyang natanggap na regalo at si eina naman ay masaya at nag pasalamat sa kanyang mga magulang dahil sa kanyang natanggap na regalo.

MAALAT :: MATAMIS

Ang lasa ng dagat ay maalat samantala ang pan de krema ay matamis.

MALINAW :: MALABO

Malinaw na nakaka-kita si eri, pero ang kanyang kapatid ay malabo ang paningin.

WAGI :: TALO

Muling na wagi ang grupo nila emy kayaʼt sila ay masayang-masaya, samantala ang grupo nila edwin ay na talo ngunit sila ay masaya pa rin.

Halimbawa ng mga Salitang Magkasalungat

Makapangyarihan ang mga salita. Ito ang nagbibigay ng puwersa upang magkaroon ng komunikasyon ang mga tao. Ang magkasalungat na mga salita ay literal na magkaiba ng kahulugan o ang mismong kabaligtaran nito. Tinutukoy nito ang kahulugan ng isang salita sa magkaiba o magkasalungat na direksyon.

Mayroon itong malaking epekto sa pagbuo ng kaisipan at sa mambabasa. Anu-ano ang mga ito? Narito ang ilan:

Puwede kang gumamit maging ng magkasalungat na mga salita upang bigyang kahulugan ang iyong nais sabihin, anupat napatitingkad ang kahulugan nito.Nagkakaroon ng kasabikan ang mambabasa.Napapalawak ang bokabularyo.Naibibigay nito hindi lamang ang kahulugan ng mga salita kundi ang damdamin ng buong pangungusap.

Para magamit ang mga magkasalungat na mga salita, nangangailangan pa din ng iba pang karagdagang salita gaya ng "hindi" upang baliktarin ang kahulugan nito at tumugma sa naunang binibigyang kahulugan.
